
# My Blog Website

## Purpose
A simple 3-page blog site showcasing posts, author info, and contact form.

## Pages
- Home: Latest posts
- About: Author info
- Contact: Contact form with validation

## Tech Stack
- HTML5
- CSS3
- JavaScript (vanilla)

## Live URL
https://Cherubismark.github.io/blog-website/

## Folder Structure
- index.html, about.html, contact.html
- /css/style.css
- /js/main.js
- /images/

## Features
- Responsive layout
- Mobile-friendly navigation
- Form validation
